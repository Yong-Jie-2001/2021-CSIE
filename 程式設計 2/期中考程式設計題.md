# 遞迴方法 vs iterative
## 遞回方法
```
終止式　==> 0!=1  1!=1
遞迴式  ==> f(n) = n*(n-1)*(n-2)*....*2*1
                 = n*f(n-1);

static void factorial(int x) {
	if (x == 1)
           return 1;
	else		
	  return (x*factorial(x-1));
	}
```

## iterative
```
public class Method1 {
	public static void main(String[] args) {
		factorial(6);
		factorial(9);
	}

	static void factorial(int x) {
		int i = x, j = 1;
		while(i > 0)
			j *= i--;
		System.out.println(x + "! = " + j);
	}
```
